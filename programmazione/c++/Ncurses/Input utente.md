<font color="red">keypad(window, bool)</font> : permette di abilitare l'utilizzo di alcuni tasti speciali nella finestra specificata, e il bool serve per attivarla o disattivarla:

| Tasto | KEY_ |
| ---- | ---- |
| -> | KEY_RIGHT |
| <- | KEY_LEFT |
| $\downarrow$ | KEY_DOWN |
| $\uparrow$ | KEY_UP |
