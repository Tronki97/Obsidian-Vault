- ## Conda
	- scaricare anaconda dal sito.
	- https://www.anaconda.com/download/success
- ## Creare il VENV
	- Virtual environment:
		- nel terminale: 
			- `conda create --name CN24`
			- `conda activate CN24`
- ## Eseguire un file
	- ### Windows:
		- andare nella directory che contiene il file da eseguire poi digitare:
			- `python "nome file.py"`
